To make a new line, use the "< br >" tag.

To italicize font, use the "< em > &< /em >" tags

To bold text, use the "< strong >< /strong > tags

Lists are a bit more tricky. You'll need to put the list within UL tags and then each
item will be wrapped in LI tags. The UL will make an unorganized (aka a bulleted list), while
the OL tag will make a numbered list. See the lists below!

The formatting for images is a bit different since we can load multiple tags within one tag.
Images open with "< img src = "" >" but there is not a closing tag for IMG. The path to the
image goes within the quotes and then it can be followed by "alt="" " to show alternative text
that can be used by screen readers. You can finally specify a height or a width, but you
shouldn't do both. You do that by typing either height or width = and then a number
(ex height = "50").
